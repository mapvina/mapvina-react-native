# Annotations

Comparison of various annotations available in MapVina React Native:

| Feature                | `CircleLayer`                                                                                                                                                | `SymbolLayer`                                                                                                                                                | `ViewAnnotation`                                                                                                                                                          | `Marker`                                                                                                                                                                          |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Can use Images         | &cross;                                                                                                                                                      | &check;                                                                                                                                                      | &cross;                                                                                                                                                                   | &cross;                                                                                                                                                                           |
| `children` RN View     | &cross;                                                                                                                                                      | iOS: Static                                                                                                                                                  | iOS: Interactive<br/>Android: Static                                                                                                                                      | Interactive                                                                                                                                                                       |
| Interactions           | Click                                                                                                                                                        | Click                                                                                                                                                        | iOS: Complete<br/>Android: Click, drag, callout                                                                                                                           | Only `children` RN View is interactive                                                                                                                                            |
| Control z-index        | &check;                                                                                                                                                      | &check;                                                                                                                                                      | iOS: Always on top<br/>Android: Layer id `com.mapvina.annotations.points`                                                                                                | Always on top                                                                                                                                                                     |
| Clustering             | &check;                                                                                                                                                      | &check;                                                                                                                                                      | &cross;                                                                                                                                                                   | &cross;                                                                                                                                                                           |
| Style with Expressions | &check;                                                                                                                                                      | &check;                                                                                                                                                      | &cross;                                                                                                                                                                   | &cross;                                                                                                                                                                           |
| Android Implementation | [`CircleLayer`](https://mapvina.com/mapvina-native/android/api/-map-libre%20-native%20-android/com.mapvina.android.style.layers/-circle-layer/index.html) | [`SymbolLayer`](https://mapvina.com/mapvina-native/android/api/-map-libre%20-native%20-android/com.mapvina.android.style.layers/-symbol-layer/index.html) | [`annotation.Symbol`](https://github.com/mapvina/mapvina-plugins-android/blob/main/plugin-annotation/src/main/java/org/mapvina/android/plugins/annotation/Symbol.java) | [`annotation.MarkerView`](https://github.com/mapvina/mapvina-plugins-android/blob/main/plugin-markerview/src/main/java/org/mapvina/android/plugins/markerview/MarkerView.java) |
| iOS Implementation     | [`MLNCircleStyleLayer`](https://mapvina.com/mapvina-native/ios/latest/documentation/mapvina/mlncirclestylelayer/)                                         | [`MLNSymbolStyleLayer`](https://mapvina.com/mapvina-native/ios/latest/documentation/mapvina/mlnsymbolstylelayer/)                                         | [`MLNAnnotationView`](https://mapvina.com/mapvina-native/ios/latest/documentation/mapvina/mlnannotationview/)                                                          | [`MLNAnnotationView`](https://mapvina.com/mapvina-native/ios/latest/documentation/mapvina/mlnannotationview/)                                                                  |
